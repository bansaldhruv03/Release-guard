Developer CLI Setup

Instructions for developers to install Release Guard locally.

1. Install Globally

npm install -g release-guard-cli.tgz

2. Authenticate

Get your API key from the Org Admin page and run:

release-guard login <YOUR-API-KEY>

3. Initialize Hooks in Repository

Navigate to your local repository directory and run:

release-guard init
